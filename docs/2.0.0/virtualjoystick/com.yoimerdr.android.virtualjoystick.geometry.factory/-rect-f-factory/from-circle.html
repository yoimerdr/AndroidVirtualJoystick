fromCircle

public final static RectF fromCircle(Circle circle)

Creates a Rect around a circle.

This method traits the circle center as the rect center and circle's radius as side distance for each rect side.

This is equal to:

val cx = circle.center.x.
val cy = circle.center.y
val side = circle.radius
val rect = RectF(cx - side, cy - side, cx + side, cy + side)
